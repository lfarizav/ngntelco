## Step-by-Step

### 1. Pull Docker Images

```bash
cd docker_open5gs

docker pull ghcr.io/herlesupreeth/docker_open5gs:master
docker tag ghcr.io/herlesupreeth/docker_open5gs:master docker_open5gs

# For UERANSIM (simulated 5G gNB + UE)
docker pull ghcr.io/herlesupreeth/docker_ueransim:master
docker tag ghcr.io/herlesupreeth/docker_ueransim:master docker_ueransim
```

### 2. Deploy the 5G SA Core

```bash
docker compose -f sa-deploy.yaml up -d
```

This starts all 5G core NFs + MongoDB + WebUI.

### 3. Add a Test 5G Subscriber

Via WebUI at `http://localhost:9999` (admin / 1423):
- IMSI: `001010000000001`
- K: `465B5CE8B199B49FAA5F0A2EE238A6BC`
- OPc: `E8ED289DEBA952E4283B54E88E6183CA`
- SST: 1 (eMBB slice)

Or via MongoDB:

```bash
docker exec -it mongo mongosh open5gs --eval '
  db.subscribers.insertOne({
    "imsi": "001010000000001",
    "security": {
      "k": "465B5CE8B199B49FAA5F0A2EE238A6BC",
      "opc": "E8ED289DEBA952E4283B54E88E6183CA",
      "amf": "8000"
    },
    "slice": [{ "sst": 1, "default_indicator": true,
      "session": [{ "name": "internet", "type": 3,
        "qos": { "index": 9, "arp": { "priority_level": 8 }},
        "ambr": { "downlink": {"value": 1, "unit": 3}, "uplink": {"value": 1, "unit": 3}}
      }]
    }]
  })
'
```

### 4. Start Simulated gNB + UE (UERANSIM)

```bash
docker compose -f nr-gnb.yaml up -d
docker compose -f nr-ue.yaml up -d
```

Or with srsRAN (ZMQ simulation):

```bash
docker compose -f srsgnb_zmq.yaml up -d
docker compose -f srsue_5g_zmq.yaml up -d
```

### 5. Verify

```bash
# Check UERANSIM UE got an IP
docker exec nr-ue ip addr show uesimtun0

# Test connectivity
docker exec nr-ue ping -c 3 -I uesimtun0 8.8.8.8

# Check AMF for registered UE
docker logs amf 2>&1 | grep -i "registered"

# Check NRF for discovered NFs
docker logs nrf 2>&1 | grep -i "profile"
```

## Key Differences from 4G

| Aspect | 4G EPC | 5G SBA |
|--------|--------|--------|
| Architecture | Point-to-point interfaces | Service-Based (HTTP/2) |
| Discovery | Static config | NRF (dynamic) |
| Authentication | EPS-AKA | 5G-AKA (SUCI/SUPI) |
| Session | Default bearer | PDU Session |
| Slicing | Not native | Native (SST/SD) |
| User Plane | GTP-U fixed | PFCP (flexible UPF) |

## Troubleshooting

| Symptom | Cause | Fix |
|---------|-------|-----|
| NFs fail to register with NRF | NRF not ready | Wait 10s, check `docker logs nrf` |
| UE registration rejected | Wrong SUCI/PLMN | Verify MCC=001, MNC=01 in `.env` |
| No PDU session | SMF-UPF N4 issue | Check `docker logs smf` for PFCP errors |
| UERANSIM can't connect | AMF N2 unreachable | Verify AMF_IP in nr-gnb.yaml matches .env |

## Cleanup

```bash
docker compose -f nr-ue.yaml down
docker compose -f nr-gnb.yaml down
docker compose -f sa-deploy.yaml down
```
